# 🇷🇺 Miniblox Language Translator inChat

Simple **transliteration script** for Miniblox.

No APIs. No requests. No external dependencies.  
Just fast client-side character mapping.

---

## ✨ Features

- 🔄 Real-time transliteration when sending chat
- ⚡ Fully offline (no translation API)
- 🧠 Keeps commands untouched (messages starting with `/`)
- 🔠 Supports uppercase and lowercase
- 🧩 Lightweight userscript (Tampermonkey / Violentmonkey compatible)

---

## 📦 Installation

### 1️⃣ Install a Userscript Manager

You’ll need one of these browser extensions:

- Tampermonkey
- Violentmonkey
- Greasemonkey

### 2️⃣ Add the Script

1. Open your userscript manager
2. Create a new script
3. Paste the script code
4. Save
5. Visit: https://miniblox.io/
